If you know the agency you want to request records from, start typing it below. Once you've located an agency, you can bookmark the page to return there any time.

You can make your FOIA request to the General Services Administration right from here.

Please bear in mind:

  • You need to include your name and contact information to file a request.
  • FOIA requests are themselves public records, so do not include any unnecessary personal information.
  • You can only request records that already exist.
  • You may be contacted by a FOIA processing officer to help target and narrow your request to something they can fulfill.
